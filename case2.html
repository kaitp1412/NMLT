Inline HTML elements

HTML defines a long list of availiable inline tags, a complete list of which can be found on the Mozilla Developer Network

  • To bold text,use < strong>
  • To italic text, use < em>
  • Abbreviations, like HTML should use < abbr > , with an optional title atribute for the full phrase
  • Citations, like - Mark Otto, should be < cite>
  • Deleted text should use < del> and inserted text should use < ins>
  • Superscripttext use < sup> and subscriptext use < sub>

Most of these elements are styled by browsers with few modifications on our past
